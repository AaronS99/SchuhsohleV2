Schuhsohle --Bitte Google Chrome verwenden--


Für die BLE Verbindung ist Google Chrome erforderlich.
Außerdem muss evtl. unter about://flags Experimental Web Platform features auf Enabled gesetzt werden.

Bei Fehlern hilfts evtl. die Seite neu zu laden.

Um die Live Übertragung zu starten, auf CONNECT klicken und dann BT05 auswählen.

Während der Live Übertragung können die Daten auch aufgezeichnet werden. Um die Aufzeichnung
zu starten, auf Aufnahmebeginn klicken, um sie zu stoppen nochmals drauf klicken. Anschließend
wird eine Datei mit den rohen Daten heruntergelanden. Mit einem Hochladen dieser bei rohe Daten Verarbeiten
werden die Daten in eine Übersichtlichere CSV Datei gewandelt.

Um die Verbindung zu stoppen auf Disconnect klicken.

Um alte Aufzeichnungen nochmals abspielen zu lassen, die CSV Datei bei Zum Abspielen hochladen.
Diese wird dort in der originalen Geschwindigkeit abgespielt.

Über Stop kann man die Wiedergabe anhalten und anschließend auch fortsetzen.
Über den Slider drüber kann man die Wiedergabegeschwindigkeit einstellen.
Beim Anzeigen des Graphens kann es, falls man nicht Chrome verwendet, zu Performance Problemen kommen
